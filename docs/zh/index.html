An overview of my software and hardware

Programming Language

XiaoXuan Language is an elegant, full-stack programming language. It consists of several variants and can be used to develop a variety applications, including digital circuits and chips, microcontroller programs, GPU shaders, native programs, system programs, service programs, web applications and more. All variants use the same syntax and design philosophy, allowing you to learn one language for all domains of development.

The implemented variants:

  • XiaoXuan Script

    Build high-performance, solid web applications that compile to WebAssembly on-the-fly without requiring any build tools.

  • XiaoXuan Core

    Build powerful user-space system programs with extremely fast startup speed and small footprint, direct syscall and interoperation with C shared libraries.

  • XiaoXuan Managed

    Build secure and robust programs for business systems, cloud-native (microservices and serverless functions), data science, A.I. and more. Each program runs in its own isolated environement, programs do not need to be installed, only a URL address is required to run them.

  • XiaoXuan Native

    A general-purpose programming language inspired by Rust, but much simpler. Automatic memory management without garbage collection, and no complex concepts such as ownership, borrow checking, and lifetimes. Supports compilation to native code for architectures such as x86-64, aarch64 and riscv64.

  • XiaoXuan Micro

    Easily build high-performance firmware for microcontroller, the built-in micro VM makes it possible to "write once, run anywhere".

  • XiaoXuan Logic

    A brand new and modern hardware description language (HDL) that allows you to develop hardware and chipss in the same way as you develop open source software. It comes with a GPU-accelerated simulator that significantly improves test efficiency.

The XiaoXuan programming language supports writing code in local languages, such as Chinese, Japanese and French. In addition to keywords, the standard library add related documentation are also translated into local languages. Using local language programming is beneficial for children, beginners and non-computer majors who are not native English speakers.

Operating System

In addition, I have designed two new operating systems, one for daily use by software engineers and the other for microcontrollers. Both are implemented using XiaoXuan Lang.

  • XiaoYu OS

    A brand new operating system designed for engineers daily use. The system is very secure, all applications run in isolated environement with lowest possible privileges, applications must request permissions from the user when they need them. The system is also very clean, and even after decades of use, it will remain as clean as new, with no junk left behind. Applications are maintained by developers, and the system does not require package maintainers.

  • XiaoYu Micro OS

    A true operating system (not RTOS) for microcontrollers (MCUs). Applications (firmwares) can be run simply by "downloading" and "configuring", without the need to install cumbersome development tools and debuggers and then compile from source code. It completely changes the way firmware is developed and deployed.

Hardware

I am sharing my open-source hardware designs here. Please note that I do not sell physical hardware, and there are no purchase links. Here you will find only the hardware schematics, design documents, and usage instructions.

  • XiaoYu Debugger

    A very simple debugger (programmer) base on CMSIS-DAP implemented using a low cost MCU STM32F042 and very few parts. Used for downloading firmware to other MCUs or debugging programs.

  • XiaoYu Dev Board Mini

    A reference board for XiaoXuan Micro development. It uses a low-cost MCU STM32G070 in LQFP package, which is easy to make and solder by yourself.

  • XiaoYu Dev Board

    A reference board for XiaoYu Micro OS development. It integrates a debugger as well as numerous interfaces, buttons and LEDs using the common MCU STM32F412.
